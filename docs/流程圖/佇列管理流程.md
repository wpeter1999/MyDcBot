# 佇列管理流程

> 佇列操作的完整流程和並行安全機制

## 佇列入隊流程

```mermaid
flowchart TD
    Start([Enqueue 呼叫]) --> Lock[取得互斥鎖<br/>mu.Lock]
    Lock --> DeferUnlock[設定 defer Unlock]
    
    DeferUnlock --> CheckCapacity{len >= capacity?}
    CheckCapacity -->|是| Unlock1[解鎖]
    CheckCapacity -->|否| Append[append 到切片]
    
    Unlock1 --> ReturnError[回傳 ErrQueueFull]
    Append --> Unlock2[解鎖]
    
    Unlock2 --> ReturnNil[回傳 nil]
    
    style Start fill:#5865F2
    style ReturnNil fill:#57F287
    style ReturnError fill:#ED4245
```

## 佇列出隊流程

```mermaid
flowchart TD
    Start([Dequeue 呼叫]) --> Lock[取得互斥鎖<br/>mu.Lock]
    Lock --> DeferUnlock[設定 defer Unlock]
    
    DeferUnlock --> CheckEmpty{len == 0?}
    CheckEmpty -->|是| Unlock1[解鎖]
    CheckEmpty -->|否| GetFirst[取得第一首歌]
    
    Unlock1 --> ReturnEmpty[回傳空值和 false]
    
    GetFirst --> CopyLeft[copy 左移元素]
    CopyLeft --> Truncate[截斷切片長度]
    Truncate --> Unlock2[解鎖]
    
    Unlock2 --> ReturnSong[回傳 song, true]
    
    style Start fill:#5865F2
    style ReturnSong fill:#57F287
```

## 快照功能流程

```mermaid
flowchart TD
    Start([Snapshot 呼叫]) --> Lock[取得互斥鎖<br/>mu.Lock]
    Lock --> DeferUnlock[設定 defer Unlock]
    
    DeferUnlock --> MakeSlice[建立新切片<br/>cap = len]
    MakeSlice --> Copy[深拷貝所有元素]
    
    Copy --> Unlock[解鎖]
    Unlock --> Return[回傳副本]
    
    style Start fill:#5865F2
    style Return fill:#57F287
```

## 並行安全機制

```mermaid
sequenceDiagram
    participant G1 as Goroutine 1
    participant Lock as Mutex
    participant Queue as Queue
    participant G2 as Goroutine 2
    
    G1->>Lock: Enqueue: Lock()
    Note over Lock: G1 獲得鎖
    
    G2->>Lock: Dequeue: Lock()
    Note over G2: 等待中...
    
    G1->>Queue: append(song)
    G1->>Lock: Unlock()
    Note over Lock: 鎖釋放
    
    Note over Lock: G2 獲得鎖
    G2->>Queue: 取出 songs[0]
    G2->>Queue: copy 和截斷
    G2->>Lock: Unlock()
    
    Note over Lock: 鎖釋放
```

## GuildPlayer 整合流程

```mermaid
flowchart TD
    Start([播放器操作]) --> Type{操作類型}
    
    Type -->|Enqueue| CheckStopped{播放器已停止?}
    Type -->|Dequeue| DirectDequeue[直接從 Queue 出隊]
    Type -->|QueueSnapshot| DirectSnapshot[直接取得快照]
    
    CheckStopped -->|是| ReturnError[回傳 ErrPlayerStopped]
    CheckStopped -->|否| CallEnqueue[呼叫 queue.Enqueue]
    
    CallEnqueue --> CheckFull{佇列已滿?}
    CheckFull -->|是| ReturnFull[回傳 ErrQueueFull]
    CheckFull -->|否| Success1[加入成功]
    
    DirectDequeue --> GetSong[取得歌曲]
    GetSong --> CheckEmpty{佇列為空?}
    CheckEmpty -->|是| ReturnEmpty[回傳 Song{}, false]
    CheckEmpty -->|否| Success2[回傳 song, true]
    
    DirectSnapshot --> Success3[回傳副本]
    
    Success1 --> End([完成])
    Success2 --> End
    Success3 --> End
    
    style Start fill:#5865F2
    style End fill:#57F287
    style ReturnError fill:#ED4245
    style ReturnFull fill:#ED4245
```

## 佇列清空流程

```mermaid
flowchart TD
    Start([Clear 呼叫]) --> Lock[取得互斥鎖<br/>mu.Lock]
    Lock --> DeferUnlock[設定 defer Unlock]
    
    DeferUnlock --> Reslice[重設切片長度為 0]
    Note1[保留底層陣列<br/>只重設長度為 0]
    
    Reslice --> Note1
    Note1 --> Unlock[解鎖]
    Unlock --> Complete([完成])
    
    style Start fill:#5865F2
    style Complete fill:#57F287
```

## 容量管理流程

```mermaid
flowchart TD
    Start([NewQueue 呼叫]) --> CheckCapacity{capacity < 0?}
    CheckCapacity -->|是| SetZero[capacity = 0]
    CheckCapacity -->|否| UseValue[使用傳入值]
    
    SetZero --> MakeSlice[make 切片]
    UseValue --> MakeSlice
    
    MakeSlice --> InitLen[len = 0]
    MakeSlice --> InitCap[cap = capacity]
    
    InitLen --> CreateQueue[建立 Queue 結構]
    InitCap --> CreateQueue
    
    CreateQueue --> Return[回傳 *Queue]
    
    style Start fill:#5865F2
    style Return fill:#57F287
```

    DeferUnlock --> GetLen[取得 len(songs)]

## 記憶體管理

```mermaid
flowchart LR
    A[初始化] --> B[建立切片<br/>長度=0, 容量=50]
    B --> C[使用中<br/>動態增長]
    
    C --> D{操作}
    D -->|Enqueue| E[len++<br/>可能擴容]
    D -->|Dequeue| F[len--<br/>保留容量]
    D -->|Clear| G[len=0<br/>保留容量]
    
    E --> H[記憶體使用]
    F --> H
    G --> H
    
    style A fill:#5865F2
    style H fill:#FEE75C
```

## 並行場景示例

```mermaid
sequenceDiagram
    participant P as Producer<br/>(Enqueue)
    participant Q as Queue
    participant C as Consumer<br/>(Dequeue)
    
    par 並行操作
        P->>Q: Enqueue(song1)
        Note over Q: Lock → Append → Unlock
        
        and
        
        C->>Q: Dequeue()
        Note over Q: 等待鎖...
    end
    
    Q-->>P: nil (成功)
    
    C->>Q: Dequeue() 繼續
    Note over Q: Lock → 取出 → Unlock
    Q-->>C: song1, true
    
    par 再次並行
        P->>Q: Enqueue(song2)
        and
        P->>Q: Enqueue(song3)
        and
        C->>Q: Dequeue()
    end
    
    Note over Q: 互斥鎖確保安全
```

## 錯誤處理流程

```mermaid
flowchart TD
    Start([佇列操作]) --> Try[嘗試操作]
    
    Try --> Check{檢查條件}
    
    Check -->|佇列已滿| E1[ErrQueueFull]
    Check -->|佇列為空| E2[回傳 false]
    Check -->|播放器已停止| E3[ErrPlayerStopped]
    Check -->|正常| Success[操作成功]
    
    E1 --> Log[記錄錯誤]
    E2 --> Handle[正常處理]
    E3 --> Log
    
    Log --> Return1[回傳錯誤]
    Handle --> Return2[回傳空值]
    Success --> Return3[回傳結果]
    
    style Start fill:#5865F2
    style Success fill:#57F287
    style E1 fill:#ED4245
    style E3 fill:#ED4245
```

## 使用範例流程

```mermaid
flowchart TD
    Start([使用者執行 /play]) --> Create[建立/取得 GuildPlayer]
    Create --> Song[取得 Song 結構]
    
    Song --> Enqueue[呼叫 Enqueue]
    Enqueue --> Check{播放中?}
    
    Check -->|是| Queue[只加入佇列]
    Check -->|否| PlayFirst[Dequeue 並播放]
    
    Queue --> Response1[回應: 已加入佇列<br/>位置: 第 N 首]
    PlayFirst --> Response2[回應: 正在播放]
    
    Response1 --> End([完成])
    Response2 --> End
    
    style Start fill:#5865F2
    style End fill:#57F287
```
