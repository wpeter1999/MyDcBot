# 下載功能流程

> 從使用者請求到檔案上傳的完整流程

## 完整下載流程

```mermaid
flowchart TD
    Start([使用者執行 /download]) --> Defer[延遲回應<br/>避免 3 秒逾時]
    Defer --> GetParams[取得參數]
    
    GetParams --> Format[format: 格式選擇]
    GetParams --> URL[url: YouTube URL]
    
    Format --> Validate{驗證參數}
    URL --> Validate
    
    Validate -->|URL 為空| Error1[❌ 請提供 YouTube 網址]
    Validate -->|有效| UpdateMsg[⏳ 正在下載...]
    
    UpdateMsg --> CreateTemp[建立暫存目錄<br/>/tmp/discord-downloads]
    CreateTemp --> GenFilename[產生唯一檔案名<br/>userID_timestamp_title.ext]
    
    GenFilename --> BuildArgs[建構 yt-dlp 參數]
    BuildArgs --> Execute[執行 yt-dlp<br/>逾時: 5 分鐘]
    
    Execute -->|失敗| Error2[❌ 下載失敗]
    Execute -->|成功| FindFile[尋找下載的檔案]
    
    FindFile -->|找不到| Error3[❌ 找不到檔案]
    FindFile -->|找到| GetSize[取得檔案大小]
    
    GetSize --> CheckSize{大小檢查}
    CheckSize -->|> 25MB| Error4[❌ 檔案太大<br/>💡 使用 Opus 或較短歌曲]
    CheckSize -->|<= 25MB| OpenFile[開啟檔案]
    
    OpenFile --> BuildResponse[建構回應訊息<br/>檔案名稱 + 大小]
    BuildResponse --> Upload[上傳到 Discord]
    
    Upload -->|失敗| Error5[❌ 上傳失敗]
    Upload -->|成功| CleanupSuccess[清理暫存檔案]
    
    Error2 --> CleanupFail[清理暫存檔案]
    Error3 --> CleanupFail
    Error4 --> CleanupFail
    Error5 --> CleanupFail
    
    CleanupSuccess --> Success([✅ 完成])
    CleanupFail --> End([結束])
    
    style Start fill:#5865F2
    style Success fill:#57F287
    style Error1 fill:#ED4245
    style Error2 fill:#ED4245
    style Error3 fill:#ED4245
    style Error4 fill:#ED4245
    style Error5 fill:#ED4245
```

## yt-dlp 參數建構流程

```mermaid
flowchart TD
    Start([開始建構參數]) --> BaseArgs[基本參數]
    
    BaseArgs --> NoPlaylist[--no-playlist]
    BaseArgs --> MaxSize[--max-filesize 25M]
    BaseArgs --> Duration[--match-filter duration < 600]
    BaseArgs --> Output[--output 輸出範本]
    
    NoPlaylist --> CheckFormat{檢查格式}
    MaxSize --> CheckFormat
    Duration --> CheckFormat
    Output --> CheckFormat
    
    CheckFormat -->|mp3-320| MP3[--extract-audio<br/>--audio-format mp3<br/>--audio-quality 0]
    CheckFormat -->|m4a-256| M4A[--extract-audio<br/>--audio-format m4a<br/>--audio-quality 256K]
    CheckFormat -->|opus-192| Opus[--extract-audio<br/>--audio-format opus<br/>--audio-quality 192K]
    CheckFormat -->|flac| FLAC[--extract-audio<br/>--audio-format flac]
    CheckFormat -->|wav| WAV[--extract-audio<br/>--audio-format wav]
    CheckFormat -->|其他| Default[預設 MP3 320kbps]
    
    MP3 --> Append[附加 URL]
    M4A --> Append
    Opus --> Append
    FLAC --> Append
    WAV --> Append
    Default --> Append
    
    Append --> Return([回傳參數陣列])
    
    style Start fill:#5865F2
    style Return fill:#57F287
```

## 格式轉換流程

```mermaid
flowchart LR
    A[YouTube 影片] --> B[yt-dlp]
    
    B --> C[提取音訊流]
    C --> D{目標格式}
    
    D -->|MP3| E[轉換為 MP3<br/>320 kbps]
    D -->|M4A| F[轉換為 M4A<br/>256 kbps]
    D -->|Opus| G[轉換為 Opus<br/>192 kbps]
    D -->|FLAC| H[轉換為 FLAC<br/>無損]
    D -->|WAV| I[轉換為 WAV<br/>原始 PCM]
    
    E --> J[輸出檔案]
    F --> J
    G --> J
    H --> J
    I --> J
    
    J --> K[檢查大小]
    K --> L{<= 25MB?}
    
    L -->|是| M[上傳]
    L -->|否| N[拒絕]
    
    style A fill:#5865F2
    style M fill:#57F287
    style N fill:#ED4245
```

## 檔案大小檢查流程

```mermaid
flowchart TD
    Start([檔案已下載]) --> Stat[os.Stat 取得資訊]
    Stat --> GetSize[fileInfo.Size]
    
    GetSize --> Compare{大小比較}
    Compare -->|> 25MB| Calc1[計算 MB: size/1024/1024]
    Compare -->|<= 25MB| Calc2[計算 MB: size/1024/1024]
    
    Calc1 --> Error[回傳錯誤<br/>檔案太大 XX.XX MB<br/>建議使用 Opus 格式]
    Calc2 --> Open[開啟檔案]
    
    Open --> PrepareMsg[準備訊息<br/>檔案名 + 大小]
    PrepareMsg --> Upload[上傳到 Discord]
    
    Upload --> Success([成功])
    
    style Start fill:#5865F2
    style Success fill:#57F287
    style Error fill:#ED4245
```

## Discord 上傳流程

```mermaid
sequenceDiagram
    participant Handler as 下載處理器
    participant File as 檔案系統
    participant Discord as Discord API
    participant User as 使用者
    
    Handler->>File: os.Open(filePath)
    File-->>Handler: file handle
    
    Handler->>Handler: 建構訊息<br/>檔案名 + 大小
    
    Handler->>Discord: UpdateInteractionResponse
    Note over Discord: Content: 訊息<br/>Files: [file handle]
    
    Discord->>Discord: 讀取檔案內容
    Discord->>Discord: 上傳到 CDN
    
    Discord-->>User: 顯示訊息 + 檔案
    Discord-->>Handler: 成功回應
    
    Handler->>File: file.Close()
    Handler->>File: os.Remove(filePath)
```

## 暫存檔案管理

```mermaid
flowchart TD
    Start([開始下載]) --> Check{暫存目錄存在?}
    Check -->|否| Create[建立目錄<br/>/tmp/discord-downloads]
    Check -->|是| GenName[產生檔案名]
    Create --> GenName
    
    GenName --> Format[userID_timestamp_title.ext]
    Format --> Download[下載到暫存目錄]
    
    Download --> Process[處理檔案]
    Process --> Upload{上傳成功?}
    
    Upload -->|是| Cleanup1[os.Remove 刪除]
    Upload -->|否| Cleanup2[os.Remove 刪除]
    
    Cleanup1 --> Success([成功清理])
    Cleanup2 --> Success
    
    style Start fill:#5865F2
    style Success fill:#57F287
```

## 錯誤處理決策樹

```mermaid
flowchart TD
    Start([發生錯誤]) --> Type{錯誤類型}
    
    Type -->|URL 無效| E1[提示: 請提供 YouTube 網址]
    Type -->|yt-dlp 失敗| E2[提示: 下載失敗 + 錯誤訊息]
    Type -->|找不到檔案| E3[提示: 找不到檔案]
    Type -->|檔案太大| E4[提示: 檔案太大<br/>建議: Opus 或較短歌曲]
    Type -->|上傳失敗| E5[提示: 上傳失敗 + 錯誤訊息]
    Type -->|逾時| E6[提示: 處理逾時<br/>建議: 選擇較短影片]
    
    E1 --> Log[記錄錯誤]
    E2 --> Log
    E3 --> Log
    E4 --> Log
    E5 --> Log
    E6 --> Log
    
    Log --> Cleanup[清理暫存檔案]
    Cleanup --> Response[回應使用者]
    
    style Start fill:#FEE75C
    style Response fill:#ED4245
```

## 逾時處理流程

```mermaid
flowchart TD
    Start([開始下載]) --> SetTimeout[設定逾時: 5 分鐘]
    SetTimeout --> Execute[執行 yt-dlp]
    
    Execute --> Race{競速}
    Race -->|正常完成| Success[下載成功]
    Race -->|逾時| Timeout[context.DeadlineExceeded]
    
    Timeout --> Kill[中止 yt-dlp 行程]
    Kill --> Error[回傳逾時錯誤]
    
    Success --> Continue([繼續處理])
    Error --> End([結束])
    
    style Start fill:#5865F2
    style Success fill:#57F287
    style Error fill:#ED4245
```
