# 播放清單處理流程

> 從檢測 URL 到完成播放的完整流程

## 完整流程圖

```mermaid
flowchart TD
    Start([使用者執行 /play URL]) --> CheckURL{檢查 URL 類型}
    
    CheckURL -->|包含 list=| IsPlaylist[判定為播放清單]
    CheckURL -->|單曲 URL| SingleSong[單曲處理流程]
    
    IsPlaylist --> Extract[執行 yt-dlp --flat-playlist]
    Extract --> ParseJSON[解析 JSON 輸出]
    
    ParseJSON --> CheckEmpty{播放清單是否為空?}
    CheckEmpty -->|是| Error1[❌ 播放清單是空的]
    CheckEmpty -->|否| BuildEntries[建立 PlaylistEntry 陣列]
    
    BuildEntries --> GetPlayer[取得 GuildPlayer]
    GetPlayer --> FirstSong[取出第一首]
    
    FirstSong --> PlayFirst[播放第一首]
    PlayFirst --> CheckSuccess{播放成功?}
    
    CheckSuccess -->|失敗| Error2[❌ 播放失敗]
    CheckSuccess -->|成功| EnqueueRest[將其餘歌曲加入佇列]
    
    EnqueueRest --> Loop[迴圈處理每首歌]
    Loop --> CreateSong[建立 Song 結構]
    CreateSong --> AddQueue[Enqueue 到佇列]
    AddQueue --> CheckMore{還有更多?}
    
    CheckMore -->|是| Loop
    CheckMore -->|否| SetCurrent[設定第一首為 CurrentSong]
    
    SetCurrent --> UpdateStatus[更新語音頻道狀態]
    UpdateStatus --> BuildMessage[建構回應訊息]
    
    BuildMessage --> ShowList[顯示前 10 首]
    ShowList --> Response[回應使用者]
    
    Response --> Success([✅ 完成])
    
    style Start fill:#5865F2
    style Success fill:#57F287
    style Error1 fill:#ED4245
    style Error2 fill:#ED4245
    style IsPlaylist fill:#FEE75C
```

## 時序圖

```mermaid
sequenceDiagram
    participant User as 使用者
    participant Handler as 指令處理器
    participant YtDlp as yt-dlp
    participant Player as GuildPlayer
    participant Lavalink
    participant Discord
    
    User->>Handler: /play 播放清單 URL
    Handler->>Handler: IsPlaylistURL() = true
    
    Handler->>YtDlp: ExtractPlaylist(url)
    Note over YtDlp: --flat-playlist --dump-json
    YtDlp-->>Handler: PlaylistEntry 陣列 (JSON)
    
    Handler->>Player: GetOrCreatePlayer()
    Player-->>Handler: guildPlayer
    
    loop 第 2 首到最後一首
        Handler->>Player: Enqueue(song)
        Player-->>Handler: nil (成功)
    end
    
    Handler->>Lavalink: JoinVoiceAndPlay(第一首)
    Lavalink-->>Handler: 播放成功
    
    Handler->>Player: SetCurrentSong(第一首)
    Handler->>Discord: UpdateVoiceChannelStatus()
    
    Handler->>User: 📋 播放清單載入成功！
    Note over User: 顯示歌曲清單
```

## yt-dlp 執行流程

```mermaid
flowchart LR
    A[播放清單 URL] --> B[yt-dlp]
    B --> C{參數}
    
    C --> D[--dump-json]
    C --> E[--flat-playlist]
    C --> F[--skip-download]
    
    D --> G[輸出 JSON 格式]
    E --> G
    F --> G
    
    G --> H[多行 JSON 輸出]
    H --> I[每行一個影片]
    
    I --> J[解析每一行]
    J --> K{解析成功?}
    
    K -->|是| L[加入 entries]
    K -->|否| M[記錄錯誤，跳過]
    
    L --> N[建構完整 URL]
    N --> O[回傳 entries 陣列]
    
    style A fill:#5865F2
    style O fill:#57F287
```

## 佇列加入流程

```mermaid
flowchart TD
    Start([entries 陣列]) --> First[第一首用於播放]
    First --> Loop{i = 1 到 len-1}
    
    Loop --> GetEntry[取得下一個 entry]
    GetEntry --> CreateSong[建立 Song 結構]
    
    CreateSong --> SetFields[設定欄位]
    SetFields --> Title[Title = entry.Title]
    SetFields --> URL[URL = entry.URL]
    SetFields --> User[RequestedBy = userID]
    
    Title --> Enqueue[guildPlayer.Enqueue]
    URL --> Enqueue
    User --> Enqueue
    
    Enqueue --> CheckError{錯誤?}
    CheckError -->|是| LogError[記錄錯誤]
    CheckError -->|否| LogSuccess[記錄成功]
    
    LogError --> Next{i < len-1?}
    LogSuccess --> Next
    
    Next -->|是| Loop
    Next -->|否| Done([完成])
    
    style Start fill:#5865F2
    style Done fill:#57F287
```

## 訊息建構流程

```mermaid
flowchart TD
    Start([entries 陣列]) --> Header[建構標題訊息]
    Header --> Count[顯示總歌曲數]
    Count --> NowPlaying[顯示正在播放]
    
    NowPlaying --> CheckCount{歌曲數量}
    CheckCount -->|<= 10| ShowAll[顯示全部]
    CheckCount -->|> 10| ShowPartial[顯示前 10 首]
    
    ShowAll --> LoopAll[迴圈處理所有歌曲]
    ShowPartial --> LoopPartial[迴圈處理前 10 首]
    
    LoopAll --> AddLine[加入歌曲行]
    LoopPartial --> AddLine
    
    AddLine --> CheckFirst{是第一首?}
    CheckFirst -->|是| AddArrow[加入 ▶️ 前綴]
    CheckFirst -->|否| AddNumber[只加入編號]
    
    AddArrow --> Append[附加到訊息]
    AddNumber --> Append
    
    Append --> CheckMoreSongs{還有更多歌曲?}
    CheckMoreSongs -->|是| AddEllipsis[加入 ... 還有 N 首]
    CheckMoreSongs -->|否| Complete[訊息完成]
    
    AddEllipsis --> Complete
    Complete --> Return([回傳訊息字串])
    
    style Start fill:#5865F2
    style Return fill:#57F287
```

## 錯誤處理流程

```mermaid
flowchart TD
    Start([開始處理]) --> Try[嘗試提取]
    
    Try --> Check1{yt-dlp 成功?}
    Check1 -->|否| Error1[回傳: 提取播放清單失敗]
    Check1 -->|是| Check2{entries 不為空?}
    
    Check2 -->|否| Error2[回傳: 播放清單是空的]
    Check2 -->|是| Check3{第一首播放成功?}
    
    Check3 -->|否| Error3[回傳: 播放失敗]
    Check3 -->|是| EnqueueLoop[加入其餘到佇列]
    
    EnqueueLoop --> CheckEnqueue{Enqueue 錯誤?}
    CheckEnqueue -->|是| LogOnly[只記錄，繼續處理]
    CheckEnqueue -->|否| Continue[繼續]
    
    LogOnly --> Continue
    Continue --> Success([成功完成])
    
    style Start fill:#5865F2
    style Success fill:#57F287
    style Error1 fill:#ED4245
    style Error2 fill:#ED4245
    style Error3 fill:#ED4245
```
